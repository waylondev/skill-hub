# Copyright & Compliance Guidelines

## General Principles

1. **Personal/Educational Use**: Generally acceptable for offline study and personal reference
2. **Commercial Use**: Requires proper licensing or explicit permission from content owner
3. **Redistribution**: Requires explicit permission from content owner
4. **Attribution**: Always credit original creator when using downloaded content

## Risk Assessment Matrix

| Content Type | Use Case | Risk Level | Action |
|--------------|----------|------------|--------|
| Educational/tutorial | Personal learning | Low | Proceed with download |
| Conference/webinar | Research | Low | Proceed with download |
| Music video | Personal collection | Medium | Warn user, require confirmation |
| Movie/TV show | Any | High | Strongly warn, require explicit confirmation |
| Premium/paid content | Any | High | Warn about potential TOS violation |
| User-generated content | Repurposing | Medium | Check license, warn user |
| Creative Commons | Any | Low | Proceed, follow CC license terms |
| Public domain | Any | Low | Proceed freely |

## Platform-Specific Considerations

### YouTube
- Content is subject a variety of licenses
- Standard YouTube videos: personal use generally acceptable
- YouTube Premium content: requires active subscription
- Creative Commons videos: follow license terms

### Bilibili (B站)
- Operates under User Generated Content (UGC) protocol
- UP主 (creators) retain copyright to their uploads
- Personal/educational use generally acceptable
- Commercial use requires explicit permission from UP主

### Twitch
- VODs may be available for limited time
- Streamer owns content rights
- Clip downloads: check streamer's policy

### Vimeo
- Creator-controlled licensing
- Check video description for license terms
- Some content requires Vimeo Pro/subscription

## Attribution Template

When downloading, include the following information:
- Original title
- Creator/author
- Source URL
- License (if known)
- Date downloaded
- Intended use

**Example**:
```
Source: "Introduction to Machine Learning" by AI Education
URL: https://youtube.com/watch?v=abc123
Downloaded: 2026-05-02
Purpose: Personal study
License: Standard YouTube License
```

## Commercial Use Warning

If user indicates commercial/business use:
1. Strongly warn about copyright implications
2. Require explicit confirmation
3. Suggest contacting content owner for permission
4. Document the download for audit trail (if enterprise policy requires)

## Fair Use Considerations

Fair use may apply in limited circumstances:
- **Criticism/Review**: Using clips for commentary
- **Education**: Using in classroom or training (check institution policy)
- **Research**: Academic analysis of content
- **News Reporting**: Using for journalistic purposes

**Note**: Fair use is a legal doctrine that varies by jurisdiction. When in doubt, seek legal advice.
